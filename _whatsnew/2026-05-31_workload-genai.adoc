=== Soporte para AWS GovCloud (US) en NetApp Workload Factory

NetApp Workload Factory admite cuentas de AWS GovCloud (US). Con una cuenta de GovCloud, puedes implementar y administrar cargas de trabajo de GenAI en las regiones de AWS GovCloud (US).
